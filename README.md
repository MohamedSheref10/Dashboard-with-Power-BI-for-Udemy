# Udemy Courses Analysis Dashboard 📊

An end-to-end Data Analysis and Visualization project built using **Power BI** to analyze Udemy courses data, uncover market trends, and extract actionable business insights regarding course ratings, enrollment numbers, and pricing strategies.

---

## 📌 Project Overview
The main goal of this project is to analyze a dataset of **1,200+ Udemy courses** to understand what drives high student enrollment and course satisfaction. The project covers the full data cycle: **Data Cleaning, Transformation (ETL), DAX Modeling, and Dashboard Design**.

### 🔍 Key Business Questions Answered:
* Which course categories and subjects generate the highest revenue and enrollment?
* Is there a correlation between course price, ratings, and student engagement?
* How do missing values or unrated courses impact the overall business analysis?

## 🛠️ Tech Stack & Tools
* **Power BI Desktop:** Dashboard design and interactive reporting.
* **Power Query (M Language):** Data cleaning, ETL, and data type unification.
* **DAX (Data Analysis Expressions):** Created custom measures for KPIs (e.g., Average Sales, Total Revenue, Categorical translations, and Handling Missing Values).

---

## 🧹 Data Cleaning & Transformation (ETL)
The raw dataset required meticulous preprocessing to ensure high data quality:
1. **Date Unification:** Standardized various date formats into a single, unified datetime structure.
2. **Handling Missing Values:** Managed nulls and missing sales figures scientifically (e.g., using averages/medians where appropriate to avoid skewing data).
3. **Categorical Translation & Standardization:** Cleaned and unified the "Category" column to ensure accurate filtering and slicing.

---

## 📈 Key Features & KPIs Implemented
* **Executive Summary KPIs:** Total Courses, Total Subscribers, Average Rating, and Estimated Revenue.
* **Course Subject Analysis:** Breakdown of performance across different domains.
* **Dynamic Slicers:** Filter by Rating, Price Range, and Publication Year to drill down into specific trends.
